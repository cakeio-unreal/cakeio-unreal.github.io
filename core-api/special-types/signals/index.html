Signals

Overview

Source Location: Signals

The following types are all defined in the following header: 

#include "CakeIO/CakeSignals.h"

Signals are a special enum type that are used by various operations that use callbacks. Signals are returned by the callbacks and allow the callbacks to influence the control flow of the associated operation.

Directory Traversal Signals

These signals are used in directory traversal operations.

ECakeSignalGuarded

This signal is used to control guarded traversals.

Enum Value Display Name Description
Continue Continue The traversal should continue to the next element.
Abort Abort An error has been encountered and the traversal should abort early.

ECakeSignalSearch

This signal is used to control search traversals.

Enum Value Display Name Description
Continue Continue The requirements for the search have yet to be satisfied and the next element should be visited.
Success Success The requirements for the search have been satisfied and the traversal should stop.
Abort Abort An error has been encountered and the search should terminate early in an unsuccessful state.

Advanced Signals

These are specialized signals used in higher level, advanced operations.

ECakeSignalGather

This signal is used in the CakeMix GatherCustom family of functions.

Enum Value Display Name Description
Select Select Include the current element and continue the traversal.
Reject Reject Exclude the current element and continue the traversal.
SelectThenStop Select Then Stop Include the current element and then stop the traversal.
RejectThenStop Reject Then Stop Exclude the current element and then stop the traversal.
Abort Abort A critical error was encountered and the gather operation should immediately abort.

ECakeSignalBatchOp

This signal is used in the CakeAsyncIO exclusive Batch Operation functions.

Enum Value Display Name Description
Continue Continue The batch operation should process the next item.
Abort Abort An error has been encountered and the batch operation should immediately abort.
