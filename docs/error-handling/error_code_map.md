## Introduction 
This section maps all possible error codes specific IO functions can return.

## FCakeFile
The following section details the possible error codes generated by **FCakeFile**.

### CreateTextFile / CreateBinaryFile

|Error Code|Associated Policy|Context|
|:---------|:----------------|:------|
|AlreadyExists|OverwriteItems|Occurs when the file already exists and the policy disallows overwriting items.|
|FailedCreatingMissingParents|MissingParents|Occurs when the destination directory has missing parents, the policy allows creation of missing parents, but the directory creation IO operations failed.|
|DestDirDoesNotExist|MissingParents|Occurs when the destination directory has missing parents and the policy disallows missing parent creation.|
|FailedOpenW|None|Occurs when a write handle could not be obtained from the operating system.|


### CreateOrWriteTextFile / CreateOrWriteBinaryFile

|Error Code|Associated Policy|Context|
|:---------|:----------------|:------|
|AlreadyExists|OverwriteItems|Occurs when the file already exists and the policy disallows overwriting items.|
|FailedCreatingMissingParents|MissingParents|Occurs when the destination directory has missing parents, the policy allows creation of missing parents, but the directory creation IO operations failed.|
|DestDirDoesNotExist|MissingParents|Occurs when the destination directory has missing parents and the policy disallows missing parent creation.|
|FailedOpenW|None|Occurs when a write handle could not be obtained from the operating system.|


### WriteTextToFile / WriteBytesToFile

|Error Code|Associated Policy|Context|
|:---------|:----------------|:------|
|DoesNotExist|None|Occurs when the file does not exist. Write is only intended for existing files, use Create otherwise.|
|FailedOpenW|None|Occurs when a write handle could not be obtained from the operating system.|


### AppendTextToFile / AppendBytesToFile

|Error Code|Associated Policy|Context|
|:---------|:----------------|:------|
|DoesNotExist|None|Occurs when the file does not exist. Append is only intended for existing files, use Create otherwise.|
|FailedOpenW|None|Occurs when a write handle could not be obtained from the operating system.|


### ReadFileAsString / ReadFileAsBytes

|Error Code|Associated Policy|Context|
|:---------|:----------------|:------|
|DoesNotExist|None|Occurs when the file does not exist.|
|FailedOpenR|None|Occurs when a read handle could not be obtained from the operating system.|


### DeleteFile

|Error Code|Associated Policy|Context|
|:---------|:----------------|:------|
|NOP|None|Occurs when the file already does not exist.|
|FileIsReadOnly|FileDelete|Occurs when the file is read only and the policy disallows deleting read only files.|
|CouldNotChangePerms|FileDelete|Occurs when the file is read only, the policy allows deleting read only files, but the operating system could not remove the read only attribute from the file.|
|FailedDelete|None|Occurs when the operating system failed deleting the file.|


### CopyFile / CopyFileAliased

|Error Code|Associated Policy|Context|
|:---------|:----------------|:------|
|NOP|None|Occurs when the destination directory is the source file's directory.|
|DoesNotExist|None|Occurs when the file does not exist.|
|AlreadyExists|OverwriteItems|Occurs when the file already exists and the policy disallows overwriting items.|
|CouldNotReplace|OverwriteItems|Occurs when a file of the same name already exists in destination directory, the policy allows overwrites, but that file could not be deleted in preparation for the copy operation.|


### MoveFile / MoveFileAliased

|Error Code|Associated Policy|Context|
|:---------|:----------------|:------|
|NOP|None|Occurs when the destination directory is the source file's directory.|
|DoesNotExist|None|Occurs when the file does not exist.|
|AlreadyExists|OverwriteItems|Occurs when the file already exists and the policy disallows overwriting items.|
|CouldNotReplace|OverwriteItems|Occurs when a file of the same name already exists in destination directory, the policy allows overwrites, but that file could not be deleted in preparation for the move operation.|
|FailedCopy|None|Occurs when the operating system failed copying the source file to the destination directory.|
|FailedDelete|None|Occurs when the operating system failed deleting the original source file.|


### RenameFile

|Error Code|Associated Policy|Context|
|:---------|:----------------|:------|
|BadFileName|None|Occurs when the submitted name is completely empty after being sanitized for illegal characters.|
|NOP|None|Occurs when the destination directory is the source file's directory.|
|DoesNotExist|None|Occurs when the file does not exist.|
|AlreadyExists|OverwriteItems|Occurs when the file already exists and the policy disallows overwriting items.|
|CouldNotReplace|OverwriteItems|Occurs when a file of the same name already exists in destination directory, the policy allows overwrites, but that file could not be deleted in preparation for the move operation.|
|FailedCopy|None|Occurs when the operating system failed copying the source file to the destination directory.|
|FailedDelete|None|Occurs when the operating system failed deleting the original source file.|




## FCakeDir
The following section details the possible error codes generated by **FCakeDir**.

### CreateDir

|Error Code|Associated Policy|Context|
|:---------|:----------------|:------|
|NOP|None|Occurs when the directory already exists.|
|FailedCreatingMissingParents|MissingParents|Occurs when the destination directory has missing parents, the policy allows creation of missing parents, but the directory creation IO operations failed.|
|DestDirDoesNotExist|MissingParents|Occurs when the destination directory has missing parents and the policy disallows missing parent creation.|
|FailedCopy|None|Occurs when the operating system failed creating the directory.|


### ExistsOrCreate

|Error Code|Associated Policy|Context|
|:---------|:----------------|:------|
|FailedCreatingMissingParents|MissingParents|Occurs when the destination directory has missing parents, the policy allows creation of missing parents, but the directory creation IO operations failed.|
|DestDirDoesNotExist|MissingParents|Occurs when the destination directory has missing parents and the policy disallows missing parent creation.|
|FailedCopy|None|Occurs when the operating system failed creating the directory.|


### CopyDir / CopyDirAliased

|Error Code|Associated Policy|Context|
|:---------|:----------------|:------|
|DoesNotExist|None|Occurs when the source directory does not exist.|
|NOP|None|Occurs when the source directory and the destination directory are the same.|
|FailedCreatingMissingParents|MissingParents|Occurs when the destination directory has missing parents, the policy allows creation of missing parents, but the directory creation IO operations failed.|
|DestDirDoesNotExist|MissingParents|Occurs when the destination directory has missing parents and the policy disallows missing parent creation.|
|FailedCopy|None|Occurs when the operating system failed copying the source directory and its contents to the destination directory.|


### MoveDir / MoveDirAliased

|Error Code|Associated Policy|Context|
|:---------|:----------------|:------|
|DoesNotExist|None|Occurs when the source directory does not exist.|
|NOP|None|Occurs when the source directory and the destination directory are the same.|
|FailedCreatingMissingParents|MissingParents|Occurs when the destination directory has missing parents, the policy allows creation of missing parents, but the directory creation IO operations failed.|
|DestDirDoesNotExist|MissingParents|Occurs when the destination directory has missing parents and the policy disallows missing parent creation.|
|FailedCopy|None|Occurs when the operating system fails copying the source directory during the move operation.|
|FailedDelete|None|Occurs when the operating system fails deleting the source directory after the copy operation has completed.|


### RenameDir

|Error Code|Associated Policy|Context|
|:---------|:----------------|:------|
|BadDirectoryName|None|Occurs when the new directory name is empty after sanitizing it for illegal characters.|
|DoesNotExist|None|Occurs when the source directory does not exist.|
|NOP|None|Occurs when the source directory and the destination directory are the same.|
|DestDirDoesNotExist|MissingParents|Occurs when the destination directory has missing parents and the policy disallows missing parent creation.|
|FailedCreatingMissingParents|MissingParents|Occurs when the destination directory has missing parents, the policy allows creation of missing parents, but the directory creation IO operations failed.|
|FailedCopy|None|Occurs when the operating system fails copying the source directory during the move operation.|
|FailedDelete|None|Occurs when the operating system fails deleting the source directory after the copy operation has completed.|


### DeleteDir

|Error Code|Associated Policy|Context|
|:---------|:----------------|:------|
|NOP|None|Occurs when the directory does not exist.|
|FailedDelete|None|Occurs when the operating system failed deleting the directory and its contents.|
