## Files
CakeIO uses a dedicated type for representing files on the filesystem. It contains interfaces to support a wide variety of common IO operations, from reading and writing files to moving and copying files. Futhermore, the CakeIO file objects also provide various utility functions for retrieving information about a file.

### File Documentation
The following documentation will provide an overview of many of the interfaces you will commonly use with CakeIO file objects. However, if you want to see the entire interface, please examine the source code which is fully documented. Each article will include the location of the header files for the associated types.
